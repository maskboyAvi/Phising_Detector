# Frontend - Phishing Detection System

React/Next.js frontend for the phishing email detection application.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Frontend runs at: **http://localhost:3000**

## Production Build

```bash
npm run build
npm start
```

## Dependencies

- Next.js 16
- React 19
- Axios
- Tailwind CSS
